Enterprise-grade RESTful API for generating customizable QR codes with guaranteed uptime and high performance.

Enterprise Deployment: This API is deployed on Northflank infrastructure with 99.99% uptime SLA and global CDN distribution.

URL QR Code Generation

Generate QR codes for website URLs with custom styling options.

POST /api/v1/qr/url
Request Body:
{
  "url": "https://example.com",
  "options": {
    "size": 10,
    "format": "PNG",
    "foreground_color": "#000000",
    "background_color": "#FFFFFF",
    "error_correction": "M"
  }
}
Example Request with Full Customization:
curl -X POST "https://site--qr-code-generator-api--lrw6bbnkrwj5.code.run/api/v1/qr/url" \
-H "Content-Type: application/json" \
-d '{
  "url": "https://northflank.com",
  "options": {
    "size": 15,
    "format": "SVG",
    "foreground_color": "#1e40af",
    "background_color": "#f0f9ff",
    "module_drawer": "rounded",
    "error_correction": "H"
  }
}'
JPEG Format Example:
curl -X POST "https://site--qr-code-generator-api--lrw6bbnkrwj5.code.run/api/v1/qr/url" \
-H "Content-Type: application/json" \
-d '{
  "url": "https://example.com",
  "options": {
    "format": "JPEG",
    "foreground_color": "#ff0000",
    "background_color": "#ffffff"
  }
}'
Success Response:
{
  "success": true,
  "data": {
    "qr_code": "data:image/png;base64,iVBORw0KGgoAAAA...",
    "content": "https://northflank.com",
    "format": "PNG",
    "options": {
      "foreground_color": "#000000",
      "background_color": "#FFFFFF",
      "size": 12,
      "border": 4,
      "error_correction": "M",
      "format": "PNG"
    }
  }
}

Customization Options

All QR code endpoints support the following customization options:

Parameter Type Default Description
size Integer 10 QR code module size (1-40)
border Integer 4 Border size around QR code (0-20)
format String "PNG" Output format: PNG, JPEG, SVG, PDF
foreground_color String "#000000" QR code color (hex format)
background_color String "#FFFFFF" Background color (hex format)
error_correction String "M" Error correction: L, M, Q, H
module_drawer String "square" Module shape: square, rounded, circle

Error Handling

The API uses standard HTTP status codes and returns detailed error messages.

Error Response Format:
{
  "success": false,
  "error": "Detailed error message"
}
Common HTTP Status Codes:
  • 200 - Success
  • 400 - Bad Request (missing required fields)
  • 404 - Endpoint not found
  • 500 - Internal server error
